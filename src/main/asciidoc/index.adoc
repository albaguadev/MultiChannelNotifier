== Data Structures
The following fields are accepted by the notification endpoint:

|===
|Field |Type |Description |Constraints

|`type`      |String |Channel to use (EMAIL, SMS, WHATSAPP) |Not Null / Must match a valid NotificationType
|`recipient` |String |Target address or phone number        |Not Blank
|`message`   |String |Content of the notification           |Not Blank
|`subject`   |String |Subject line (used by Email channel)  |Optional
|===
