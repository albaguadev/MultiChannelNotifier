== Data Structures
The following fields are required for a successful notification dispatch:

|===
|Field |Type |Description |Constraints

|`type` |String |Channel to use (EMAIL, SMS) |Not Null / Must match Strategy
|`recipient` |String |Target address or phone number |Not Blank
|`message` |String |Content of the notification |Length (1-500)
|===
